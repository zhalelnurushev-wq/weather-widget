# Погода — виджет для Android (в стиле Samsung Weather)

## Что умеет
- Виджет 4×2: часы (обновляются сами), дата, город, текущая температура с иконкой и прогноз на 5 дней
- Полупрозрачный «стеклянный» фон (#59101823, скругление 26dp) + тени у текста — читается на любых обоях
- Автоопределение геолокации (FusedLocationProvider + Geocoder для названия города)
- Обновление каждые 30 минут через WorkManager
- Нажатие на виджет открывает приложение: почасовой прогноз на 24 ч, прогноз на 5 дней, фон-градиент меняется под погоду (ясно/облачно/дождь/снег/гроза/ночь)
- Данные: Open-Meteo — бесплатно, без API-ключа и регистрации

## Сборка
1. Открыть папку проекта в Android Studio (Hedgehog или новее)
2. Дождаться Gradle Sync (потребуется интернет; при первом запуске скачается Gradle 8.7)
3. Build → Build APK или Run на подключённом телефоне

## Первый запуск
1. Открыть приложение → разрешить доступ к геолокации («При использовании приложения» достаточно)
2. Долгое нажатие на рабочий стол → Виджеты → «Погода» → перетащить на экран

## Примечания
- Если фоновое обновление не подхватывает новое местоположение (Android ограничивает геолокацию в фоне),
  виджет использует последние сохранённые координаты. Чтобы фоново обновлялась и геолокация,
  выдайте в настройках приложения «Разрешить в любом режиме» (background location).
- Иконка запуска — векторная (солнце за облаком), можно заменить в манифесте.
